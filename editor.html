Annotation editor

Jsvee supports the following actions:

  • show - moves n steps forward without pausing between the steps but the animation is visible
  • skip - moves n steps forward without showing the steps at all

Jsvee also supports the following actions in buttons:

  • runToLine - runs the animation to line n without pausing between the steps but the animation is visible
  • skipToLine - runs the animation to line n without showing the steps at all
